# Semana 1 - PI_REDES_SISTEMAS OPERATIVOS

## Descripción

En esta semana se desarrollan tres programas principales:

- **IPv4**: implementación de comunicación usando sockets con IPv4.
- **IPv6**: implementación de comunicación usando sockets con IPv6.
- **Sim**: simulación protocolo

## Uso del Makefile

El `makefile` permite compilar, ejecutar y limpiar el proyecto de forma automática.

### Compilacion

- `make`: Compila todo
- `make build_IPv4`: Compila el código de IPv4.cpp
- `make build_IPv6`: Compila el código de IPv6.cpp
- `make build_Sim`: Compila el código de la simulación, de sim.cpp

### Ejecutar

- `make run_IPv4`: Ejecuta el código de IPv4.cpp
- `make run_IPv6`: Ejecuta el código de IPv6.cpp
- `make run_Sim`: Ejecuta el código de la simulación

### Limpieza

- `make clean`: Limpia el directorio `bin` donde se contienen todos los ejecutables
